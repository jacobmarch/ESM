# Tactical FPS Esports Manager

A text-based esports management simulation game where you manage a roster of players for a 5v5 tactical FPS game similar to Counter-Strike or Valorant.

## Getting Started

1. Ensure you have Python 3.7+ installed on your system.
2. Clone this repository.
3. Navigate to the project directory.
4. Run the game using: `python main.py`

## Features (Planned)

- Manage a roster of players for a tactical FPS esports team
- Scout and recruit new players
- Train and develop player skills
- Arrange and participate in matches against AI-controlled teams
- Manage team finances and contracts
